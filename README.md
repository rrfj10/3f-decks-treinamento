# 3f-decks-treinamento

## Ambiente

Copie `.env.example` para `.env` e ajuste a chave do gerador:

```text
GENERATOR_API_KEY=<sua-chave-local>
```

O arquivo `.env` fica fora do Git.

O gerador permite configurar até 5 conexões de LLM pela própria tela.
Use o botão com ícone de plug no topo do gerador para salvar um slot no `.env`.
